# TanStack Query

Tuyau provides first-class integration with TanStack Query for React and Vue applications. Integrations are designed to be a simple and thin wrapper around TanStack Query's Options API.

## Overview

The TanStack Query integrations provide:
- **Type-safe query options** - Generate properly typed query options for `useQuery`, `useSuspenseQuery`
- **Type-safe infinite query options** - Generate properly typed options for `useInfiniteQuery`
- **Type-safe mutation options** - Generate properly typed mutation options for `useMutation`
- **Automatic key management** - Automatically generate and manage query keys for cache management
- **Full TypeScript inference** - Complete type inference for inputs and outputs
